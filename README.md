# CAT :3

A minimalistic custom shell for Windows with playful ASCII branding and practical automation.  
CAT-Shell combines essential system commands, shortcuts, and a built-in downloader (`CAT-DF`) into one lightweight console.

---

## ✨ Features
- Minimal design: 300~ lines of Python, no bloat
- Colorized output with `colorama`
- ASCII cat branding and playful `:3` signature
- Fast shortcuts for everyday tasks
- Built-in downloader for popular apps
- Practical automation (shutdown, process management, file operations)

---

## 🐱 Commands

### CAT commands
- `cat info` — Displays system information
- `cat exit` — Exits the CAT terminal
- `cat clear` — Clears the terminal screen
- `cat go` — Changes the current directory
- `cat fc` — Creates a new folder
- `cat del` — Deletes a file or folder
- `cat ls` — Lists contents of the current directory
- `cat fmf` — Creates a full folder path
- `cat rf` — Reads contents of a file
- `cat rn` — Renames files or folders
- `cat mv` — Moves files or folders
- `cat kill` — Kills process by ID or name
- `cat pl` — Shows process list
- `cat goto` — Finds a file name in the current directory

### Fast shortcuts
- `bw` — Fast Browser
- `ed` — Fast Notepad
- `sd` — Fast shutdown
- `run` — Fast run program
- `dd` — Fast Dodi repacks (Original)
- `fg` — Fast Fitgirl repacks (Original)
- `play` — Fast play music
- `df` — Fast download file in current directory (`hl df` for more info)
